Was ist Directory Busting?

Directory Busting ist eine Technik, mit der versteckte Verzeichnisse oder Dateien auf einem Webserver aufgespürt werden. Sie wird von Administratoren verwendet, um Schwachstellen zu identifizieren, aber auch von Angreifern, die nach ungeschützten Daten suchen. Tools wie Gobuster durchsuchen systematisch eine Liste potenzieller Verzeichnisnamen und vergleichen diese mit den auf dem Server vorhandenen Verzeichnissen.

In dieser Simulation lernen Sie, wie Directory Busting funktioniert. Ziel ist es, versteckte Verzeichnisse auf einem Server zu finden. Am Ende müssen Sie die gefundenen Verzeichnisse analysieren und das Zielverzeichnis manuell im Browser öffnen, um herauszufinden, was sich darin befindet.

Nutze den folgenden Befehl, um den Scan korrekt zu starten:

gobuster dir -u http://192.168.0.10 -w wordlist.txt
